# portfolio
Работа для учебной практики. Выполнила: студент 1 курса, Грязнова Сабина Романовна, группы ИСиП-25-11-1

Работа представляет собой личный сайт-портфолио, где я делюсь своим творчеством: рисунками, стихами и фотографиями.


  **Как запустить?**
  
1. Скачать папку с проектом.
2. Найти файл index.html
3. Открыть его двойным кликом (или через браузер: Файл -> Открыть)
4. Готово.


  **Что внутри?**
  
1. index.html # Главная страница (приветствие + навигация)
2. about.html # Страница "Обо мне"
3. drawings.html # Галерея рисунков (сетка 4×2)
4. photos.html # Фотографии (сетка 2×2)
5. poems.html # Стихотворения
6. style.css # Все стили: цвета, шрифты, анимации
7. script.js # Интерактив: клики, модальные окна, анимации
8. README.md # Этот файл
9. drawings/ # Мои рисунки (8 работ)
10. images/ # Персонажи и фото для страницы "Обо мне" (4 рисунка персонажа и 2 фотографии)
11. photos/ # Фотографии, сделанные мной (4 фотографии)
12. visual/ # Визуальный макет интерфейса (мои наброски дизайна (5 экранов)


  **Что умеет сайт?**
  
1. Навигация:
- Кнопки на главной ведут на нужные страницы.
- На каждой странице есть кнопка для возврата.

2. Галерея рисунков и фото:
- Картинки выстроены аккуратной сеткой.
- Клик по изображению открывает на весь экран модальное окно с изображением.
- Под каждой работой имеется подпись с датой.
- При наведении карточка немного "всплывает".

4. Стихотворения:
- Список стихов с названиями и датами.
- Клик по строке с названием раскрывает текст.
- Стрелочка "▶" меняется на "▼" при открытии.

5. Дизайн и анимации
- Плавное появление элементов при прокрутке
- Анимация "покачивания" у персонажей
- Градиентный фон в сине-фиолетовых тонах
- Адаптивность: сайт красиво выглядит и на телефоне, и на компьютере


  **Шрифты**    (Подключены через Google Fonts)

- Montserrat - для основного текста
